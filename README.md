## 🛠️ Development

<div align="center">

**Ready to contribute? Get started locally:**

<br>

</div>

```bash
# 📥 Clone the repository
git clone git@github.com:garage44/expressio.git
cd expressio

# 📦 Install dependencies
bun install

# 🚀 Start development server
cd packages/expressio
bun run dev
```
